To set up the enviorment:
    1. Download UV from: https://docs.astral.sh/uv/getting-started/installation/
    2. run uv sync from main repository directory
    3. go into the front end directory and run npm install
    4. run npm run dev.
    5. activate the venv that uv creates
    6. go into the backend directory and run fastapi dev main.py
    7. Connect to localhost page (given by npm run dev).
